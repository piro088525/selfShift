# selfShift プロジェクト

CTO のための資格取得ロードマップと学習記録を管理するリポジトリ。

## 学習セッションの運用ルール

### 学習中

- ユーザーが分からない箇所を質問してきたら、簡潔に解説する
- 間違えた問題や曖昧な知識は `weak-points.md` に記録する

### 学習終了時

1. チャットでその日の学習内容を振り返り、記憶定着を促す
2. `study-log.md` に日付・学習内容・学習時間・振り返りを追記する
3. 間違えた問題があれば `weak-points.md` に追記する（間違えた問題 / 曖昧な知識 / 試験直前チェックリスト）

### ファイルの役割

| ファイル | 用途 |
|---|---|
| `career-roadmap.md` | 全体の資格取得ロードマップ（4〜6年計画） |
| `*/README.md` | 各資格の目標・学習方針 |
| `*/study-log.md` | 学習記録（日付ごとの内容と振り返り） |
| `*/weak-points.md` | 間違えた問題・曖昧な知識・試験直前チェックリスト |
| `*/basics.html` | 基礎知識のビジュアルまとめ（ブラウザ閲覧用） |
