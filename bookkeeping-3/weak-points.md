# 弱点・復習ノート

間違えた問題、曖昧な知識、不安な箇所を記録する。試験直前の総復習に使用。

## 間違えた問題

### [04-21] 小切手と手形の混同
- **問題**: 小切手で支払った場合の勘定科目
- **自分の回答**: 支払手形
- **正解**: 当座預金
- **なぜ間違えたか**: 小切手と手形を混同した
- **ポイント**: 小切手 → 当座預金から引き落とされる。手形 → 支払手形（別物）

### [04-22] 内金を仕入と回答
- **問題**: 商品を発注し内金を現金で払った場合の勘定科目
- **自分の回答**: 仕入
- **正解**: 前払金（資産）
- **なぜ間違えたか**: 内金の段階では商品をまだ受け取っていない → 仕入にならない
- **ポイント**: 内金・手付金 → 前払金。仕入になるのは商品を受け取った時

### [04-23] 小切手 → 当座預金（再ミス）
- **問題**: 小切手で支払った場合の勘定科目
- **自分の回答**: （小切手と当座預金を混同）
- **正解**: 当座預金
- **なぜ間違えたか**: 2回目のミス。定着していない
- **ポイント**: 小切手は当座預金口座から引き落とされる。**小切手 = 当座預金**

### [04-24] 支払利息の借方・貸方
- **問題**: 借入金の支払利息を当座預金口座から引き落とした場合の仕訳
- **自分の回答**: 支払利息を貸方（右）に記入
- **正解**: 支払利息は費用 → 借方（左）
- **なぜ間違えたか**: 支払利息が費用であることの判断に迷った
- **ポイント**: 「支払○○」は基本的に費用 → 借方

### [04-25] 小切手 → 当座預金（4回目のミス）
- **問題**: 商品を仕入れ、代金は小切手を振り出して支払った
- **自分の回答**: 貸方を「現金」と回答
- **正解**: 貸方は **当座預金**
- **なぜ間違えたか**: 4回目。完全に定着していない
- **ポイント**: **自分が振り出した小切手 = 自分の当座預金から引き落とされる**。問題集の該当ページに付箋必須

### [04-25] 現金過不足 vs 雑損の使い分け
- **問題**: 月末の現金実査で帳簿より ¥4,000 不足、原因不明
- **自分の回答**: 借方を「雑損」と回答
- **正解**: 借方は **現金過不足**
- **なぜ間違えたか**: 雑損に直接振ってしまった
- **ポイント**: 「原因不明」が出てきた段階では一時勘定の **現金過不足** で受ける。**決算時**にも原因不明のままなら、そこで初めて雑損/雑益に振り替える

### [04-25] 仕入返品の処理
- **問題**: 仕入商品の一部を返品、買掛金から控除
- **自分の回答**: 貸方を「仕入返品」と回答
- **正解**: 貸方は **仕入**
- **なぜ間違えたか**: 「仕入返品」という独立勘定があると思い込んだ（選択肢のダミーに引っかかった）
- **ポイント**: 3級に「仕入返品」勘定はない。**仕入を貸方に書く = 費用のマイナス** で表現する

### [04-25] 前期貸倒の回収
- **問題**: 前期に貸倒処理した売掛金 ¥40,000 のうち ¥18,000 を現金で回収
- **自分の回答**: 貸方を「売掛金」と回答
- **正解**: 貸方は **償却債権取立益**（収益）
- **なぜ間違えたか**: 前期に消した売掛金がまだ存在していると勘違い
- **ポイント**: 前期に貸倒処理した時点で売掛金は帳簿から消えている → 貸方に立てられない。**当期分の回収 → 貸倒引当金戻入**、**前期分の回収 → 償却債権取立益**（別物）

### [04-26] 前払金・前受金の判断軸
- **誤解**: 「商品の前払いは前受金、その他は前払金」と覚えていた
- **正解**: 区別は **「自分が払ったか／受け取ったか」** で決まる（商品か否かは関係ない）
- **ポイント**:
  - 前払金（資産）= 自分が**払った**側 → 後で商品/サービスをもらえる
  - 前受金（負債）= 自分が**受け取った**側 → 後で商品/サービスを渡す義務
  - 「前払」「前受」という日本語の通りに覚える。**主語は常に自分（自社）**

### [04-26] 立替金の方向
- **誤解**: 「立替金は会社が払うべきお金を従業員が立て替えた時」と覚えていた（逆向き）
- **正解**: 立替金は **会社が他者のために立て替えた** 場合の勘定（資産・後で回収する権利）
- **ポイント**:
  - 立替金 = 会社→他者 方向（会社が出した）→ 資産
  - 従業員立替金 = 立替金の中で相手が従業員の場合（性質は同じ）
  - 逆方向（会社が借りた状態）→ **預り金**（負債）または **未払金**（負債）

### [04-27] 当期発生の売掛金が貸倒れた場合
- **ルール**: 当期に発生した売掛金が貸倒れた → **貸倒引当金は使えない**
- **正解の仕訳**: (借) **貸倒損失** / (貸) 売掛金
- **理由**: 貸倒引当金は前期末時点の債権リスクへの備え。当期生まれの売掛金には設定されていない
- **判断フロー**:
  - 前期以前の売掛金 + 引当金あり → 引当金充当
  - 当期の売掛金 → 貸倒損失で全額処理
  - 引当金が不足 → 引当金 + 不足分を貸倒損失（合わせ技）

### [04-27] 評価勘定（マイナスの資産）の動き
- **対象**: 貸倒引当金、減価償却累計額
- **ルール**: 普通の資産と動きが**逆**
  - 増えるとき → **貸方**（右）
  - 減るとき → **借方**（左）
- **イメージ**: 資産にくっつく「影」。設定時は右に生まれ、取り崩し時は左に書いて消す
- **混乱しやすい点**: 「資産が減る = 貸方」と覚えていると逆になる。評価勘定は資産の「マイナス」なので普通資産と挙動が反転

### [04-27] 経過勘定の締切：PLは損益、BSは繰越
- **誤解しやすい点**: 受取家賃と前受家賃で締切方法が違う理由が分からない
- **正解**:
  - **PL項目（受取家賃などの収益・費用）**→ 損益勘定に振替えてゼロリセット。前期繰越なし
  - **BS項目（前受家賃などの資産・負債・純資産）**→ 次期繰越→前期繰越で継承
- **覚え方**: 「**PLは損益でサヨナラ、BSは繰越でまた明日**」
- **追加知識**: 「次期繰越」「前期繰越」は仕訳ではなく、T勘定の左右を揃えるための事務的記入

## 曖昧な知識

### 小切手 vs 手形
| | 小切手 | 手形 |
|---|---|---|
| 勘定科目 | **当座預金** | **支払手形 / 受取手形** |
| いつ払う？ | すぐ（振り出した時点） | 将来の期日に払う |
| イメージ | 即時決済 | 後払いの約束 |

### 商品に関係あるか否かで勘定科目が変わる
| | 商品に関係ある | 商品に関係ない（土地・車など） |
|---|---|---|
| 買った | 仕入 | 土地・車両運搬具・建物など |
| 後払い | 買掛金 | **未払金** |
| 前払い | 前払金 | 前払金 |

### 付随費用の扱い
- 資産の取得時の手数料・運送費など → 取得原価に含める（別の費用にしない）
- リフォーム → 価値を高める部分は「建物」、現状維持は「修繕費」
