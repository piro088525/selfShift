| 項目 | 時間 |

| 一般的な必要時間 | 50〜70h |

## 学習スタイル

**問題先行型**。テキストは通読せず辞書的に使う。

```
1. 問題を解く（分からなくてOK）
    ↓
2. 解説を読む（ここがインプット）
    ↓
3. 理解できない → テキストの該当箇所だけ読む
    ↓
4. 次の問題へ
```

## ディレクトリ構成

```
bookkeeping-3/
├── README.md        … 目標・学習方針（このファイル）
├── basics.html      … 第2問攻略の前提知識（帳簿・補助簿・伝票）
├── study-log.md     … 学習記録（日付・内容・振り返り）
└── weak-points.md   … 間違えた問題・曖昧な知識・試験直前チェックリスト
```

## 学習の流れ

```
1. 問題集（書籍）で学習
    ↓
2. 分からない箇所をチャットで質問
    ↓
3. 回答内容を weak-points.md に記録
    ↓
4. 学習終了時にチャットで振り返り → 記憶定着
    ↓
5. study-log.md に記録
    ↓
6. 試験直前に weak-points.md を総復習
```
