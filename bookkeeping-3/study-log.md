- **振り返り**: 借方＝資・費、貸方＝負・純・収。資産＝負債＋純資産。左右が必ず一致する複式簿記の原則。

## 2026-04-19（第2回）

- **学習内容**: 仕入・売上の仕訳、貸付金の仕訳、貸倒れの処理
- **学習時間**: 30分
- **振り返り**:
  - 仕入（費用）は常に借方、売上（収益）は常に貸方。反対側に「どうやって？」を入れる
  - 貸付金は「返してもらう権利」＝資産の増加 → 借方。勘定科目の漢字に惑わされず5要素で判断する
  - 貸倒引当金：前期の売掛金には引当金を取り崩す、当期の売掛金は貸倒損失（費用）で処理
  - 引当金の目的は「利益を正しい期間に対応させる」（保守主義の原則）

## 2026-04-20（第3回）

- **学習内容**: 租税公課、収入印紙
- **学習時間**: 30分
- **振り返り**:
  - 租税公課（そぜいこうか）＝日常的な税金・公的手数料の勘定科目（費用）
  - 収入印紙＝紙の契約書・領収書に貼る税金の証紙。電子契約なら印紙税は不要
  - 収入印紙を購入 → 租税公課（借方）/ 現金（貸方）

## 2026-04-21（第4回）

- **学習内容**: 商品売買の仕訳（仕入・買掛金 vs 固定資産・未払金）、前払金、小切手と手形の違い
- **学習時間**: 30分
- **振り返り**:
  - 業務に関係ある商品 → 仕入 / 買掛金。業務に関係ない（車など）→ 車両運搬具 / 未払金
  - 手付金 = 前払金（資産）。商品を受け取った時に前払金を取り崩す
  - 小切手で支払い → 当座預金（即時決済）。手形 → 支払手形（将来払いの約束）← 間違えた

## 2026-04-22（第5回）

- **学習内容**: 小切手の受取、商品券、内金（前払金）、仕入返品
- **学習時間**: 30分
- **振り返り**:
  - 他社振出の小切手を受け取った → 銀行で現金化できるので「現金」として扱う
  - 商品券で代金を受け取った → 受取商品券（資産）
  - 内金を払った段階では仕入にならない → 前払金（資産）← 間違えた
  - 仕入返品 → 貸方に仕入（仕入の取消 = 費用の減少）

## 2026-04-23（第6回）

- **学習内容**: 現金過不足、付随費用、修繕と改良
- **学習時間**: 30分
- **振り返り**:
  - 現金の実際有高が帳簿より多い → 借方に現金、貸方に現金過不足
  - 資産の取得に付随する費用（売買手数料など）→ 取得原価に含める（別の費用にしない）
  - リフォーム → 資産価値を高める部分は「建物」、維持・修理は「修繕費」（費用）に分ける
  - 小切手 → 当座預金をまた間違えた（要反復）

## 2026-04-24（第7回）

- **学習内容**: 借入金の利息、未払費用、土地の購入
- **学習時間**: 30分
- **振り返り**:
  - 借入金の支払利息は費用 → 借方。当座預金から引き落とし → 貸方に当座預金 ← 間違えた
  - 未払水道光熱費 → 負債。まだ払っていない費用を計上する
  - 土地の購入 → 借方に土地（資産）、貸方は買掛金ではなく未払金（商品以外の後払い）
  - 基本等式の再確認：資産＝負債＋純資産。費用は左、収益は右
