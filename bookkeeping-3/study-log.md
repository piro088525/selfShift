## 2026-04-26（第9回）

- **学習内容**: 立替金・従業員立替金、取得原価に含める付随費用、前払金・前受金の整理
- **学習時間**: 約3時間
- **振り返り**:
  - **立替金 vs 従業員立替金**: どちらも「**会社が**立て替えた」方向で資産。従業員立替金は相手が従業員の場合の呼び名。性質は同じ
    - ※ 逆方向（従業員が会社のお金を立て替えた）は立替金ではなく、未払金などになる
  - **取得原価に付随費用を含める**: 取得原価 = 本体価格 + 付随費用
    - 固定資産（土地・建物・備品）→ 仲介手数料・登記費用・運送費・据付費を含める
    - 仕入 → 引取運賃（当社負担分）を含める
    - 理由: 「使える状態」にするまでの費用は原価の一部（取得原価主義）
  - **前払金・前受金の判断軸**（重要修正）: 「商品 vs 商品以外」ではなく「**払ったか受け取ったか**」で決まる
    - 前払金（資産）= 自分が**払った**側 → 後で商品/サービスをもらえる
    - 前受金（負債）= 自分が**受け取った**側 → 後で商品/サービスを渡す義務
  - 「車や建物は商品か」: 自社にとって**販売目的なら商品**、**使用目的なら固定資産**（例: 中古車販売店の車は商品、社用車は車両運搬具）

## 2026-04-27（第10回）

- **学習内容**: 経過勘定の本質、再振替仕訳の意味、損益振替、貸倒引当金の充当、評価勘定（マイナスの資産）、試算表の原理、基礎用語の整理
- **学習時間**: 約3時間（深掘り）
- **振り返り**:
  - **PL項目とBS項目の締切の違い**（最重要）
    - 収益・費用（PL）→ 期末に「損益」勘定へ振替て**ゼロリセット**。翌期は前期繰越なし、ゼロから再スタート
    - 資産・負債・純資産（BS）→ 残高を**「次期繰越」**として書き、翌期に**「前期繰越」**として継承
    - 受取家賃に前期繰越がなく、前受家賃に前期繰越があるのはこの違いから
    - 覚え方: **「PLは損益でサヨナラ、BSは繰越でまた明日」**
  - **「次期繰越」「前期繰越」は仕訳ではない**: 帳簿の左右を揃えるための事務的な書き込み。仕訳帳には登場せず、T勘定にだけ書く
  - **再振替仕訳の意味（発生主義）**: 前期末の経過勘定を期首に逆仕訳で戻すのは、「サービスを提供した期間の収益・費用」として正しく計上するため（期間損益計算）
    - もし発生主義を使わないと、前期は何もしてないのに売上計上、当期は働いてるのに売上ゼロという不正確な成績表になる
    - 「同じ金額が左右にぶつかってゼロになる」のは、古い記録を消して新しい年度のデータに書き換える操作
  - **損益振替の意味**: 収益・費用は1年間のスコアボード。期末に「損益」勘定で集計して純利益／純損失を確定し、繰越利益剰余金（純資産）へ移して翌期へ繰り越す
    - 損益勘定は決算日に1日だけ立ち上がる集計用の箱。借方=費用、貸方=収益
  - **貸倒引当金の充当ルール**（仕訳が3パターンに分岐）
    - 前期発生の売掛金 + 引当金あり → (借) 貸倒引当金 / (貸) 売掛金
    - **当期発生の売掛金** → (借) **貸倒損失** / (貸) 売掛金（**引当金は使えない**）
    - 引当金が不足 → 引当金 + 不足分を貸倒損失で合わせ技
    - 理由: 引当金は前期末時点の債権リスクへの備え。当期生まれの売掛金には備えがない
  - **評価勘定（マイナスの資産）**: 貸倒引当金、減価償却累計額
    - 資産にくっつく「影」。資産の真の価値 = 本体 − 評価勘定
    - 普通の資産と動きが**逆**: 増えるとき**貸方**、減るとき**借方**
    - 直接資産を減らさず別勘定にするのは、取得原価とマイナス評価を分けて表示の透明性を保つため
  - **試算表の原理（貸借平均）**: すべての勘定の借方合計 = すべての勘定の貸方合計
    - 個別勘定の左右は一致しない（残高がある）
    - 特定の2勘定だけ合算しても一致しない（相手科目が漏れる）
    - 全勘定を集めると必ず一致 = 複式簿記のチェック機能
  - **評価勘定の設定は「費用」で充当する**
    - 貸倒引当金の設定: (借) **貸倒引当金繰入**（費用）/ (貸) 貸倒引当金（評価勘定）
    - 減価償却の計上: (借) **減価償却費**（費用）/ (貸) 減価償却累計額（評価勘定）
    - 評価勘定は現金が動かず、費用を計上することで生まれる
  - **建物の修繕の判断**: 価値が上がるか、現状維持かで分かれる
    - 価値を高める部分（資本的支出）→ **建物勘定**（資産の増加）
    - 現状維持の修理（収益的支出）→ **修繕費勘定**（費用）
  - **基礎用語の確認**:
    - 主要簿（仕訳帳・総勘定元帳）と補助簿（補助記入帳・補助元帳）
    - 3伝票制 = **入金伝票・出金伝票・振替伝票** の3種類で構成
